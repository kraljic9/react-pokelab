# 🧪 PokeLab - Pokémon Discovery Tool

A comprehensive Pokémon encyclopedia powered by the **RESTful PokéAPI**. This project features a clean, laboratory-themed interface where users can search, filter, and explore detailed statistics of the Pokémon world.

## 🚀 Key Features

- **Live API Integration:** Fetches real-time data for over 1000+ Pokémon from the PokéAPI.
- **Smart Search:** Instant search functionality to find specific Pokémon by name or ID.
- **Statistical Breakdown:** Displays detailed base stats (HP, Attack, Defense, etc.) and type advantages.
- **Infinite Scroll / Pagination:** Optimized loading of large data sets for a smooth browsing experience.

## 🛠️ Technical Highlights

- **Asynchronous Logic:** Mastered `async/await` and the `Fetch API` to handle multiple concurrent data requests.
- **Complex Data Mapping:** Transformed nested API response objects into clean, readable UI components.
- **Dynamic Styling:** Implemented conditional CSS to change UI colors based on the Pokémon's primary type (e.g., Fire, Water, Grass).

## 🧠 Key Learnings

The core challenge was managing the sheer volume of data returned by the API. I focused on **performance optimization**, ensuring that images and stats load efficiently without slowing down the browser. This project significantly improved my ability to work with **external data sources** and **asynchronous programming**.
